Snake Engine

게임 엔진

SNAKE_ENGINE은 제가 직접 설계·구현한 경량 C++ 게임 엔진으로, OpenGL 4.6 기반 2D 렌더링, 입력, 사운드, 텍스트, 충돌, 상태 관리를 하나의 프레임워크로 통합했습니다. 모듈형(의존성/태그 기반) 구조라 프로젝트에 필요한 시스템만 선택적으로 가져올 수 있으며, 런타임 글리프 베이킹, 배칭 & 인스턴싱, 프러스텀 컬링/공간 해싱 등 최적화 기법을 지원합니다.

제가 현재 근무 중인 DigiPen Academy에서는 학생들에게 C++ 프레임워크를 .lib 파일 형태로만 제공하고 있습니다. 이 때문에 성능 최적화가 불가능하고, 기능 추가도 어려우며, 디버깅 역시 제한적이라는 문제를 직접 경험했습니다. 이러한 한계를 개선하기 위해, 언제든 내부 소스를 들여다보고 자유롭게 수정할 수 있는 엔진을 처음부터 직접 설계하고 개발하게 되었습니다.

엔진은 학습·디버깅 용이성을 중시해 공개 헤더/리소스 레지스트리 구조로 설계했습니다. 샘플 프로젝트를 통해 렌더링 파이프라인과 충돌 시스템, 텍스트 렌더링을 즉시 확인할 수 있고, 정적 라이브러리로 빌드해 외부 프로젝트에 연결하는 것도 가능합니다.

주요 기능
  • 상태 전환 프레임워크: Load/Init/Update/Draw/Free/Unload 단위 관리
  • 렌더링 최적화: 동일 Mesh/Material 배칭, 인스턴싱
  • 카메라 & 프러스텀 컬링: Camera2D 기반 화면 외부 오브젝트 컬링
  • 텍스트 엔진: FreeType 런타임 글리프 베이킹(정렬/멀티라인 지원)
  • 충돌 시스템: AABB/원형 콜라이더 + Spatial Hash Grid + 그룹/마스크
  • 사운드: miniaudio 기반 재생/일시정지/태그 단위 제어
  • 비동기 로딩: 백그라운드 리소스 로딩과 진행률 추적
  • 디버그 드로우 & 로거: 충돌 시각화, 통합 로그 레벨
아키텍처 개요
  • WindowManager — GLFW/GLAD 초기화, 이벤트, 스왑/클리어
  • InputManager — 키/마우스, 스크롤, 월드 좌표 변환
  • SoundManager — 사운드 로딩/플레이/인스턴스 관리
  • RenderManager — 리소스 레지스트리, 배칭/인스턴싱, 디버그 드로우
  • StateManager — GameState 전환/업데이트/드로우
  • ObjectManager — 오브젝트 수명/충돌/렌더 제출

현재도 지속적으로 업데이트 중이며, 샘플 프로젝트를 통해 실제 게임 씬에서의 렌더링·충돌·텍스트·사운드 동작을 검증했습니다.
